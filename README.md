LectoPlay

LectoPlay es una plataforma educativa diseñada para ayudar a niños con dislexia a mejorar sus habilidades
lectoras mediante juegos interactivos, actividades visuales y un entorno accesible. Utiliza colores suaves,
tipografías claras y una interfaz sencilla para hacer que el aprendizaje sea cómodo, amigable y sin presión.

El objetivo de LectoPlay es convertir la lectura en una experiencia divertida y progresiva. Cada actividad está
pensada para reforzar aspectos clave relacionados con la dislexia, como el reconocimiento de letras, la
comprensión, la memoria visual y la conciencia fonológica.
La plataforma también incluye a Panda Play, un asistente virtual que guía al usuario, explica actividades y 
ayuda cuando hay dudas.

De que tratan los juegos ?

• Lectura Rápida
El niño lee frases cortas y responde preguntas sencillas. Cada categoría (animales, escuela, comida, etc.) 
fortalece la comprensión y mejora la habilidad de leer con intención. El avance depende del rendimiento, permitiendo
que aprendan a su ritmo.

• Palabras y Colores
Se muestra una palabra y el niño debe escoger el color correcto. 
Este juego refuerza la memoria visual, el reconocimiento de palabras y la asociación entre significados y elementos visuales.

• Encuentra la Letra
Aparece una palabra y se pide identificar una letra específica dentro de ella.
Es ideal para reforzar el reconocimiento de grafemas, mejorar la concentración y aumentar la agilidad visual.

• Escribe la Palabra
El sistema da una descripción sencilla (por ejemplo: “Animal que hace miau”) y el niño debe escribir la palabra correcta.
Esto ayuda a practicar ortografía, comprensión y relación entre ideas y palabras.

• Palabra o No Palabra
Se muestran palabras reales mezcladas con palabras inventadas.
El niño debe identificar cuáles son correctas, mejorando la conciencia fonológica y la habilidad de distinguir estructuras lingüísticas.

- Características Principales
* Actividades diseñadas especialmente para niños con dislexia.
* Interfaz accesible: colores pastel, elementos suaves y botones grandes.
* Retroalimentación inmediata para reforzar el aprendizaje.
* Lectura en voz alta (TTS) para escuchar instrucciones y ejercicios.
* Asistente virtual (Panda Play) para apoyo constante.
* Diseño inclusivo y apto para aprendizaje autónomo.

Misión: 
Ofrecer herramientas digitales accesibles y atractivas que ayuden a niños con dislexia a desarrollar sus habilidades 
lectoras de forma divertida, guiada y sin miedo a equivocarse.

Visión :
Ser una plataforma educativa reconocida por transformar la lectura en un proceso motivador, creativo y lleno de confianza.

Valores:
Inclusión · Empatía · Innovación · Motivación
